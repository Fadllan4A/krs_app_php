# Aplikasi KRS Sederhana (PHP + MySQL + Bootstrap)

Proyek ini merupakan aplikasi sederhana untuk mengelola data:
- Mahasiswa
- Mata Kuliah
- KRS (mengambil matakuliah)

## Fitur
- CRUD Mahasiswa
- CRUD Mata Kuliah
- Tampilan Data KRS (JOIN antara mahasiswa dan matakuliah)
